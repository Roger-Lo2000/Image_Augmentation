# Image_augmentation  
**Simple Image Augmentation using openCV python**  
* Contain:  
   *  Vertical Motion blur  
   *  Horizontal Motion blur  
   *  light  
   *  Gaussian noise
## Example Usage  
`git clone https://github.com/Roger-Lo2000/Image_augmentation.git`  
  
`python main.py`
  
or with arguments 
  
`python main.py -k 11 -a 0.6 -m 0 -v 0.1`
